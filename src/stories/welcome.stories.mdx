# Creditdesign Svelte Components

Welcome to the `creditdesign-svelte-components` storybook.

This a collection of [svelte](https://svelte.dev/) components used to build basic layouts.

## Installation

To install the components from npm:

```bash
npm install --save-dev creditdesign-svelte-components
```

## Useage

Import each component into a svelte file with:

```js
import { Stack, Switcher, etc... } from "creditdesign-svelte-components";
```

## Load global CSS

The static CSS file can be copied into your project:

```bash
cat node_modules/creditdesign-svelte-components/static/css/index.css \
	> public/css/index.css
```
